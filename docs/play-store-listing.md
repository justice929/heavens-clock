### 자세한 설명 (최대 4000자)
```
천국의 시계(Heaven's Clock)는 생산성 앱이 아닙니다.
"시간을 세는 것"이 아니라 "오늘을 사는 것"에 집중하게 돕는 라이프 클락입니다.

■ 라이프 클락
당신이 바라는 수명을 기준으로 남은 시간을 연·일·시로 보여줍니다.
살아 있는 링과 함께, 지금 이 순간이 줄어들고 있다는 사실을 조용히 일깨웁니다.

■ 오늘의 버킷
미루고 싶지 않은 일 하나를 메인 화면에 둡니다.
거창한 목표가 아니라, 오늘 할 수 있는 한 걸음에 집중합니다.

■ 행동 알림
"남은 시간"이 아니라 "어떻게 보낼지"를 묻습니다.
- 아침: 오늘 가장 중요한 1가지 정하기
- 오후: 미룬 일 10분만 시작하기
- 저녁: 오늘 시간을 어떻게 썼는지 돌아보기

■ 홈 화면 위젯
남은 시간과 매일 바뀌는 문장을 홈 화면에서 바로 봅니다.

■ 시계 테마
무료 기본 테마부터, 고전적·미래적·럭셔리한 프리미엄 테마까지.
같은 메시지를 나만의 감각으로 바꿉니다.

■ 프리미엄
- 모든 프리미엄 시계 테마
- 임팩트 모드
- 버킷리스트 무제한
- 구매 복원 / 동기화
- 홈 화면 위젯 (디지털 카운터)
연간 또는 평생 이용권으로 한 번에.

천국의 시계는 죽음을 기억하고,
그 기억을 집중으로,
집중을 행동으로 바꾸는 작은 의식입니다.

오늘이 사라지기 전에, 오늘을 사세요.
```

## 4. 그래픽 자산 규격 (업로드용)

| 자산 | 규격 | 필수 | 메모 |
|------|------|------|------|
| 앱 아이콘 | 512 × 512 PNG (32비트, 알파) | ✅ | `icon.svg` 기반으로 내보내기 |
| 그래픽 이미지(피처) | 1024 × 500 PNG/JPG | ✅ | 검정 배경 + "Memento Mori" + 시계 링 |
| 휴대전화 스크린샷 | 최소 2장 (권장 4~8장) | ✅ | 1080×1920 등 16:9 또는 9:16 |
| 7인치 태블릿 | 선택 | ⬜ | 없어도 출시 가능 |
| 10인치 태블릿 | 선택 | ⬜ | 없어도 출시 가능 |

### 스크린샷 추천 구성 (폰 4장)
1. 메인 라이프 클락 (년·일·시 + 링)
2. 오늘의 버킷 화면
3. 홈 화면 위젯 (년·일·시 + 명언)
4. Premium 화면 (기능 6개)

> 스크린샷은 실기기에서 캡처(전원+볼륨↓) 하거나, 에뮬레이터에서 캡처.

## 5. 출시 단계 (Play Console)

1. **스토어 등록정보** — 위 1·2·4 내용 입력/업로드
2. **콘텐츠 등급** — 3번 설문 다시 작성 → 새 등급 확인
3. **프로덕션 → 새 버전 만들기**
   - AAB 업로드: `android/app/build/outputs/bundle/release/app-release.aab` (versionCode 8 / 1.0.7)
   - 출시 노트 입력 (아래 6번)
4. **출시 검토 → 게시** → 심사 (보통 12~48시간)
